## 0.2.0

- **Multi-Language AI Support**: The `suggest` command now detects your machine's locale and provides AI reasoning in your native language.
- **Language Override**: Added `--lang` (or `-l`) flag to manually specify the language for AI suggestions.
- **Automated CI/CD**: Integrated GitHub Actions for automated testing and secure OIDC-based publishing to pub.dev.
- **Auto-Tagging**: Automated version tagging on `master` pushes to streamline the release process.
