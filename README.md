# Dart Package Manager (DPM)

A stunning visual dependencies manager for Dart and Flutter. 

## Features

- Analyze your Flutter and Dart project dependencies.
- Beautiful, interactive CLI to see which packages are outdated.
- Security vulnerability checks.
- One-click interactive updates.

## Installation

```bash
dart pub global activate dart_package_manager
```

## Usage

Run the tool inside any Dart or Flutter project:

```bash
dpm
```

Then follow the interactive prompt to update your dependencies!
